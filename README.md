# ASRS Warehouse Simulation

A discrete-event simulation of an Automated Storage and Retrieval System (ASRS) developed using Python and SimPy. This project compares the performance of **FIFO (First-In, First-Out)** and **Deferred Commitment** task dispatching strategies in a multi-robot warehouse environment.

## Project Overview

The simulation models an ASRS warehouse consisting of storage racks, input/output stations, and autonomous robots. It evaluates how different task assignment strategies affect warehouse performance by measuring metrics such as waiting time, travel distance, completion time, and queue length.

## Features

- ASRS warehouse simulation
- Multi-robot task execution
- FIFO dispatching strategy
- Deferred Commitment dispatching strategy
- Storage and retrieval task simulation
- Performance comparison through graphs and tables
- Interactive dashboard using Streamlit

## Tech Stack

- Python
- SimPy
- Streamlit
- Pandas
- Matplotlib
- Visual Studio Code

## Performance Metrics

The simulation compares both strategies using:

- Average waiting time
- Average completion time
- Average travel distance
- Total travel distance
- Simulation makespan
- Average queue length
- Maximum queue length
- Number of completed tasks
